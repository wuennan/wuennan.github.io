文件查找

find命令可以根据不同的条件来进行查找文件，可根据文件名称、文件大小、文件修改时间、属主属组、权限、等方式查找相应的文件。

查找 地区 妹纸 18-25岁 约？

按名称查找name

# 在/etc目录下查找名称以ifcfg开头的文件
[root@ennan ~]# find /etc -name "ifcfg*"
# 在/etc目录下查找名称以iFC开头的文件，不区分大小写
[root@ennan ~]# find /etc/ -iname "iFC*"

按文件大小查找size

# 在/etc下查找大于5M的文件
[root@ennan ~]# find /etc/ -size +5M
# 在/etc下查找小于5M的文件
[root@ennan ~]# find /etc/ -size -5M
# 在/etc下查找大小为5M的文件（四舍五入）
[root@ennan ~]# find /etc/ -size 5M

按文件类型查找type

# f为文件
[root@ennan ~]# find /dev/ -type f
# d为目录
[root@ennan ~]# find /dev/ -type d
# l为连接
[root@ennan ~]# find /dev/ -type l
# b为块设备
[root@ennan ~]# find /dev/ -type b
# c为字符设备
[root@ennan ~]# find /dev/ -type c
# s为套接字
[root@ennan ~]# find /dev/ -type s
# p为管道文件
[root@ennan ~]# find /dev/ -type p

按时间查找mtime 如图，假设当前日期是10号，向前推6天（不包括当前日期)为4号，4号以前的日期用+6表示，4号之后的日期用-6表示，4号用6表示。

[root@localhost ~]# date

[root@localhost ~]# for i in {1..31}; do date -s "2019/08/$i" ; touch file-$i ; done

[root@localhost ~]# find ./ -type f -mtime 6

按用户和组进行查找

# 查找属主是ennan的文件
[root@localhost ~]# find /home/ -user ennan
# 查找属组是ennan的文件
[root@localhost ~]# find /home/ -group ennan
# 查找属主是ennan，属组也是ennan的目录
[root@localhost ~]# find /home/ -type d -user ennan -group ennan
# 查找没有属主的文件
[root@localhost ~]# find /home/ -nouser
# 查找没有属组的文件
[root@localhost ~]# find /home/ -nogroup
# 查找没有属主和属组的文件
[root@localhost ~]# find / -nouser -nogroup

处理动作

常规处理

查找到一个文件后，需要对文件进行如何处理？

动作 含义
-ls 以长格式显示的方式打印查找到的内容
-exec 后面跟自定义 shell 命令(标准写法 -exec ;)
查找/var/log/类型是文件的，并且名称是.log结尾的，并且7天以前的,然后删除。 
[root@localhost ~]# find /var/log/ -type f -name "*.log" -mtime +7 | xargs rm -f
# 首选，速度最快
[root@localhost ~]# find /var/log/ -type f -name "*.log" -mtime +7 -exec rm -rf {} \;
# 次之，速度相对较慢
[root@localhost ~]# find /var/log/ -type f -name "*.log" -mtime +7 -delete
# 不推荐，删除查找到的文件(仅能删除空目录)

记得文件的内容是什么，但是不清楚文件名称是什么，也不知道路径在哪，怎么办?

[root@localhost ~]# find /etc/ -type f | xargs grep "log_group" --color=auto
/etc/audit/auditd.conf:log_group = root

符号 作用
-a 与
-o 或
！ 非，同-not
查找/home目录下，属主不是root的所有文件。 
[root@localhost ~]# find /home ! -user root -type f | xargs ls -l

查找/home目录下，属主属于ennan，并且大小大于1k的文件。

[root@localhost ~]# find /home -user ennan -size +1k -type f | xargs ls -lh

练习

  1. 查找/var目录下，属主不是root，且文件名不以f开头的文件。

    [root@localhost ~]# find /var ! -user root ! -name "f*" -type f

  2. 查找/var目录下属主为root，且属组为mail的所有文件。

    [root@localhost ~]# find /var -user root -group mail -type f

  3. 查找/var目录下不属于root、lp的所有文件。

    [root@localhost ~]# find /var/ ! -user root ! -user lp -type f

  4. 查找/var目录下最近一周内其内容修改过，同时属主不为root，也不是postfix的文件。

    [root@localhost ~]# find /var/ -mtime -7 ! -user root ! -user postfix -type f

  5. 查找/etc目录下大于1M且类型为普通文件的所有文件。

    [root@localhost ~]# find /etc/ -type f -size +1M

  6. 将/etc/中的所有目录(仅目录)复制到/tmp下，目录结构不变。

    [root@localhost ~]# find /etc/ -type d | xargs -i mkdir -p /tmp/{}
# -i 表示 find 传递给xargs的结果 由{}来代替
或者
[root@localhost ~]# find /etc/ -type d -exec mkdir -p /tmp/{} \;

  7. 将/etc目录复制到/var/tmp/目录下，并查找/var/tmp/etc中的所有权限为777的目录和权限666的文件。

    [root@localhost ~]# cp /etc/ /var/tmp/ -rp
[root@localhost ~]# find /var/tmp/etc/ -type d | xargs -i chmod 777 {}
[root@localhost ~]# find /var/tmp/etc/ -type f | xargs -i chmod 666 {}
或
[root@localhost ~]# find /var/tmp/etc/ -type d -exec chmod 777 {} \;
[root@localhost ~]# find /var/tmp/etc/ -type f -exec chmod 666 {} \;

  8. 保留/var/log/下最近7天的日志文件，其他全部删除。

    [root@localhost ~]# find /var/log/ -mtime +7 -type f | xargs rm -rf

  9. 创建touch file{1..10}10个文件， 保留file9，其他一次全部删除。

    [root@localhost ~]# find ./ ! -name "file9" -name "file*" | xargs rm -rf
